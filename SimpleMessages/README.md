## 🇬🇧 English Version

### Project Overview
This project demonstrates how to build a basic translation chain using **LCEL (LangChain Expression Language)** and instantly deploy it as a production-ready REST API. By leveraging **FastAPI** and **LangServe**, we transform a local Python script into a web service that can handle HTTP requests, providing automatic Swagger documentation and standardized AI endpoints.

### Code-by-Code Explanation

#### 1. Imports and Environment Setup
```python
from dotenv import load_dotenv
from langchain_groq import ChatGroq
from langchain_core.messages import HumanMessage, SystemMessage
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from fastapi import FastAPI
from langserve import add_routes

load_dotenv()
```
*   **dotenv**: Loads the `GROQ_API_KEY` to authenticate with the Groq service.
*   **LangChain Modules**: We import components to build our LLM pipeline (model, parsers, prompts).
*   **FastAPI & LangServe**: These libraries are used to wrap our AI logic into a fast, modern web API without writing boilerplate routing code.

#### 2. Model Initialization
```python
model = ChatGroq(model="llama-3.3-70b-versatile", temperature=0.6)
```
*   Initializes the `llama-3.3-70b-versatile` model. 
*   **`temperature=0.6`**: We use a moderate temperature. Since translation requires strict adherence to grammar but natural-sounding phrasing, 0.6 offers a balance between precision and linguistic creativity.

#### 3. Prompt Engineering
```python
system_promt = "Translate the following into {language}"
promt_tamplate = ChatPromptTemplate.from_messages(
    [
        ("system" , system_promt), 
        ("user", "{text}")
    ]
)
```
*   **`ChatPromptTemplate`**: We define two input variables here: `{language}` (target language) and `{text}` (the content to translate). This dynamic templating system prevents us from hardcoding tasks and makes the API widely reusable.

#### 4. LCEL Pipeline Connection
```python
parser = StrOutputParser()

chain = promt_tamplate | model | parser
```
*   **`StrOutputParser`**: Extracts just the raw text content from the complex metadata object returned by the model.
*   **`|` (LCEL)**: The pipeline pipe operator connects the logic from left to right. The user's input flows into the prompt template, then to the LLM, and finally through the parser.

#### 5. Exposing as a REST API
```python
app = FastAPI(
    title="Simple Messages API",
    description="An API that translates text into a specified language using a language model.",
    version="1.0.0"
)

add_routes(
    app, 
    chain,
    path="/chain"
)
```
*   **`FastAPI`**: Initializes the main web application framework.
*   **`add_routes`**: The most crucial LangServe feature. Rather than manually writing endpoint functions (`@app.post`), LangServe automatically converts our `chain` logic into `POST /chain/invoke` and `/chain/stream` API endpoints with built-in validation schemas.

#### 6. Server Execution
```python
if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="localhost", port=8000)
```
*   **`uvicorn`**: A lightning-fast ASGI localized server that boots up our application on `http://localhost:8000`. You can visit `http://localhost:8000/chain/playground` to interactively test the API via a web UI automatically generated by LangServe.

## 🇹🇷 Türkçe Versiyon

### Proje Özeti
Bu proje, **LCEL (LangChain Expression Language)** kullanarak yapay zeka destekli bir çeviri zinciri oluşturmayı ve bunu **FastAPI** ile **LangServe** kullanarak hızla bir REST API'ye dönüştürmeyi amaçlar. Normal bir Python komut dosyasını, başka uygulamaların (web, mobil) istek atıp kullanabileceği dışa açık, dokümantasyonları otomatik oluşturulmuş bir "Web Servisi" haline getirmeyi gösterir.

### Kod Analizi: Neyin Neden Yapıldığı

#### 1. Kütüphanelerin Yüklenmesi
```python
from dotenv import load_dotenv
from langchain_groq import ChatGroq
from langchain_core.messages import HumanMessage, SystemMessage
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from fastapi import FastAPI
from langserve import add_routes

load_dotenv()
```
*   **dotenv**: Yerel çevresel değişkenlerden Groq API anahtarını yükler.
*   **LangChain**: Modeli, girdi-çıktı ayarlarını (parser) ve yapay zeka ile konuşma kalıplarımızı (prompt) içe aktarıyoruz.
*   **FastAPI & LangServe**: Yazdığımız yapay zeka mantığını standart web API standartlarına çevirecek kütüphanelerdir.

#### 2. Model Yapılandırması
```python
model = ChatGroq(model="llama-3.3-70b-versatile", temperature=0.6)
```
*   Daha yetenekli ve doğal ifade yapısı sunan `llama-3.3-70b-versatile` modeli seçildi.
*   **`temperature=0.6`**: Çeviri görevi (translation) spesifik kuralları olan ama diğer yandan da dilin doğallığını gerektiren bir iştir. Bu yüzden katı kurallar ile hafif yaratıcılık arasında denge sağlayan "0.6" ısı ayarı kullanılmıştır.

#### 3. Prompt Mühendisliği ve Şablonlama
```python
system_promt = "Translate the following into {language}"
promt_tamplate = ChatPromptTemplate.from_messages(
    [
        ("system" , system_promt), 
        ("user", "{text}")
    ]
)
```
*   **`ChatPromptTemplate`**: Dinamik ve tekrar kullanılabilir bir görev şablonu oluşturduk. Sisteme iki giriş değişkeni belirttik: `{language}` (çevrilmesi istenilen dil) ve `{text}` (çevrilecek metin).

#### 4. LCEL Boru Hattı (Pipeline) 
```python
parser = StrOutputParser()

chain = promt_tamplate | model | parser
```
*   **`StrOutputParser`**: Modeller arkada devasa meta veriler (token süreleri vb.) döndürür. Bu ayrıştırıcı, modelden gelen sonuçlardan sadece "temiz düz metni (string)" süzüp çıkarmaya yarar.
*   **`|` (LCEL)**: Boru hattı operatörüdür. Sistem soldan sağa akar; Kullanıcının API'den girdiği veriler sırasıyla şablonun içine girer, birleştirilip modele iletilir ve modelin cevabı da parser'dan geçip temizlenerek dışarı atılır.

#### 5. Sistemi REST API'ye Dönüştürme
```python
app = FastAPI(
    title="Simple Messages API",
    description="An API that translates text into a specified language using a language model.",
    version="1.0.0"
)

add_routes(
    app, 
    chain,
    path="/chain"
)
```
*   **`FastAPI`**: Uygulamanın web sunucusu iskeletini ayağa kaldırır ve isimlendirir.
*   **`add_routes`**: LangServe kütüphanesinin devrimsel özelliğidir. Kurduğumuz tüm LangChain mantığını (chain) sıfırdan "yönlendirme/routing" kodu yazmaya gerek kalmadan doğrudan uç noktalara (endpoint) bağlar. (Örn: `/chain/invoke` şeklinde API istekleri atılabilir hale gelir.)

#### 6. Sunucunun Başlatılması
```python
if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="localhost", port=8000)
```
*   **`uvicorn`**: Yazdığımız FastAPI web sunucusunu bilgisayarımızdaki "8000" portundan dış dünyaya açan ASGI server altyapısıdır.
*   **İpucu:** Kodu çalıştırdıktan sonra tarayıcınızdan `http://localhost:8000/chain/playground` adresine giderseniz, LangServe'in sizin için hazırladığı hazır kullanıcı arayüzü sayesinde kodunuzu tarayıcıdan görsel olarak test edebilirsiniz!
